# 🌟 Vridha - Elderly Care Management System

Vridha is a Dynamic Web Application designed to manage and support elderly care services efficiently. It provides features like pension tracking, administrator approvals, and secure user authentication.

## 🛠️ Tech Stack Used
- **Backend:** Java, Servlets, JSP
- **Database:** PostgreSQL
- **Server:** Apache Tomcat
- **Frontend:** HTML, CSS, JavaScript, Bootstrap

## 🚀 Key Features
- **Admin Dashboard:** Manage and approve/reject elderly care applications.
- **Pension Management:** Track pension status and distribution.
- **Secure Authentication:** OTP-based verification and role-based login (Admin/User).
- **Database Integration:** Secure and robust data management with PostgreSQL.

## 💻 How to Run This Project Locally
1. Clone this repository or download the ZIP file.
2. Setup a PostgreSQL database and import the database schema.
3. Open the project in Eclipse or IntelliJ IDEA.
4. Configure Apache Tomcat Server (v9.0 or above).
5. Run the project on the server and open `http://localhost:8080/vridha` in your browser.
